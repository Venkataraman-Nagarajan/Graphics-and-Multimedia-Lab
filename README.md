# Graphics and Multimedia Lab

These programs are written during the graphics and multimedia lab course.

## Execution

```bash
g++ [FILENAME] -o [OUT-VARIABLE].out -lGL -lGLU -lglut && ./[OUT-VARIABLE].out 
```
